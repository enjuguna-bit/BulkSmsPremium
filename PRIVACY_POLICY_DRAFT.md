## Summary
BulkSmsPremium helps users send, receive, and manage SMS messages, including bulk sends
from imported CSV/Excel files.

## Information We Access
- SMS/MMS content and metadata to display conversations and send messages.
- Contact names and phone numbers for contact selection and search.
- SIM subscription information to support multi-SIM sending.
- Files the user explicitly selects (CSV/Excel) for bulk messaging.
- App settings and message templates stored locally.

## How We Use Information
- Provide SMS send/receive features and inbox management.
- Personalize messages and manage bulk campaigns.
- Show contact names and improve message organization.
- Check subscription status when premium features are used.

## Sharing
- We do not sell personal data.
- Data is shared only with service providers that you enable (for example, Firebase services),
  and only as needed for app functionality.

## Data Storage And Retention
- Message data and contacts are processed locally and stored in app-private storage.
- Data remains on the device unless the user clears app data or deletes conversations.

## Security
- Data is protected by the Android app sandbox.
- Network requests should use HTTPS.

## Your Choices
- You can revoke permissions at any time in system settings.
- You can clear app data or uninstall the app to remove local data.
